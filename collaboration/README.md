# Collaboration Folder

This folder facilitates coordination between the Research Agent and Trading Agent teams.

## Purpose

- Share TODOs and progress updates
- Communicate requests between teams
- Flag warnings and critical issues
- Track data pipeline status
- Document API changes and dependencies

## Files

- **PRIORITIZED_BACKLOG.md** - **START HERE** - Prioritized tasks for both teams
- **RESEARCH_AGENT_TODO.md** - Research Agent team tasks and progress (legacy)
- **TRADING_AGENT_TODO.md** - Trading Agent team tasks and progress (legacy)
- **REQUESTS.md** - Inter-team requests and questions
- **WARNINGS.md** - Critical issues, breaking changes, and alerts
- **DATA_STATUS.md** - Current status of data pipeline and availability

## How to Use

1. Update your team's TODO file when starting/completing tasks
2. Add requests in REQUESTS.md when you need something from the other team
3. Post warnings in WARNINGS.md for any breaking changes or issues
4. Check this folder regularly for updates from the other team

## Teams

- **Research Agent**: Data pipeline, forecasting models, Databricks infrastructure
- **Trading Agent**: Trading strategy, backtesting, risk management
